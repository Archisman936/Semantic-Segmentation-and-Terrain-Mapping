## 🔭 Project Overview

This project builds a **terrain mapping pipeline** using semantic segmentation. The entire workflow is:

1. **Simulate** a barren landscape environment using Falcon AI by Duality AI
2. **Capture** synthetic terrain images from the simulator
3. **Generate** segmentation masks using a pre-trained SegFormer model
4. **Fine-tune** the SegFormer-B0 model on the captured dataset and generated masks
5. **Infer & Visualize** the segmentation results — including semantic overlays and terrain-only outputs

The model is trained to recognize and classify different terrain features such as earth, sand, rock, vegetation, sky, and other landscape elements commonly found in barren and desert environments.

#### Step 5 — Generate Segmentation Masks

This step uses the pre-trained SegFormer-B0 (trained on ADE20K) to generate pixel-level segmentation masks for each terrain image.

> ⚠️ **Update the paths** below to match your Google Drive structure.

```python
import os
import torch
from PIL import Image
from transformers import SegformerImageProcessor, SegformerForSemanticSegmentation

dataset_path = "/content/drive/MyDrive/CaptureSensor_BP_C_0_Color"  # ← YOUR dataset path
mask_path = "/content/drive/MyDrive/masks"                           # ← YOUR masks path
os.makedirs(mask_path, exist_ok=True)

processor = SegformerImageProcessor.from_pretrained("nvidia/segformer-b0-finetuned-ade-512-512")
model = SegformerForSemanticSegmentation.from_pretrained("nvidia/segformer-b0-finetuned-ade-512-512")
model.eval()

files = sorted(os.listdir(dataset_path))

for img_name in files:
    img_path = os.path.join(dataset_path, img_name)
    image = Image.open(img_path).convert("RGB")
    inputs = processor(images=image, return_tensors="pt")
    with torch.no_grad():
        outputs = model(**inputs)
    pred = torch.argmax(outputs.logits, dim=1)[0].cpu().numpy()
    Image.fromarray(pred.astype("uint8")).save(os.path.join(mask_path, img_name))

print("✅ Masks generated!")
```

#### Step 6 — Fine-Tune the SegFormer Model

This is the core training step. It fine-tunes SegFormer-B0 on your terrain dataset with the generated masks.

> ⚠️ **Update all three paths** below to match your Google Drive.

```python
import os
import torch
from torch.utils.data import Dataset, DataLoader
from PIL import Image
from transformers import SegformerForSemanticSegmentation, SegformerImageProcessor

image_dir = "/content/drive/MyDrive/CaptureSensor_BP_C_0_Color"  # ← YOUR dataset path
mask_dir  = "/content/drive/MyDrive/masks"                        # ← YOUR masks path
save_dir  = "/content/drive/MyDrive/segformer_model"              # ← YOUR save path

os.makedirs(save_dir, exist_ok=True)

processor = SegformerImageProcessor.from_pretrained("nvidia/segformer-b0-finetuned-ade-512-512")
model = SegformerForSemanticSegmentation.from_pretrained("nvidia/segformer-b0-finetuned-ade-512-512")

class SegDataset(Dataset):
    def __init__(self, image_dir, mask_dir, processor):
        self.images = sorted(os.listdir(image_dir))
        self.image_dir = image_dir
        self.mask_dir = mask_dir
        self.processor = processor

    def __len__(self):
        return len(self.images)

    def __getitem__(self, idx):
        img_name = self.images[idx]
        image = Image.open(os.path.join(self.image_dir, img_name)).convert("RGB")
        mask  = Image.open(os.path.join(self.mask_dir, img_name))
        inputs = self.processor(
            images=image,
            segmentation_maps=mask,
            return_tensors="pt"
        )
        return {
            "pixel_values": inputs["pixel_values"].squeeze(),
            "labels": inputs["labels"].squeeze()
        }

dataset = SegDataset(image_dir, mask_dir, processor)
loader = DataLoader(dataset, batch_size=2, shuffle=True)

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model.to(device)

optimizer = torch.optim.AdamW(model.parameters(), lr=5e-5)

model.train()

for epoch in range(3):
    for batch in loader:
        pixel_values = batch["pixel_values"].to(device)
        labels = batch["labels"].to(device)

        outputs = model(pixel_values=pixel_values, labels=labels)
        loss = outputs.loss

        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

    print(f"Epoch {epoch+1}, Loss: {loss.item()}")

model.save_pretrained(save_dir)
processor.save_pretrained(save_dir)

print("✅ Training + Processor saved successfully!")
```

**Training Results (Reference):**
| Epoch | Loss |
|-------|------|
| 1 | 0.8652 |
| 2 | 0.8948 |
| 3 | 0.2911 |

## 🔍 Reviewing Model Outputs

After training, use the following code blocks (the **last two cells** in the notebook) to load the fine-tuned model and visualize its segmentation predictions.

### Inference & Visualization

This code loads the fine-tuned model from your Google Drive, runs prediction on a test image, and displays a 4-panel visualization:

1. **Original** — The raw terrain image
2. **Segmentation (Semantic)** — Color-coded segmentation mask
3. **Overlay** — Semi-transparent segmentation overlaid on the original image
4. **Terrain Only (Hybrid Final)** — Final terrain extraction result

> ⚠️ **Update the paths** to point to your fine-tuned model and test image locations on Google Drive.

```python
import os
import torch
import numpy as np
from PIL import Image
from transformers import SegformerForSemanticSegmentation, SegformerImageProcessor
import matplotlib.pyplot as plt

# ← Load your fine-tuned model (update path to YOUR save directory)
save_dir = "/content/drive/MyDrive/segformer_model"
processor = SegformerImageProcessor.from_pretrained(save_dir)
model = SegformerForSemanticSegmentation.from_pretrained(save_dir)
model.eval()

# ← Load a test image (update path to YOUR dataset)
img_path = "/content/drive/MyDrive/CaptureSensor_BP_C_0_Color/0000050.png"
image = Image.open(img_path).convert("RGB")

# Run inference
inputs = processor(images=image, return_tensors="pt")
with torch.no_grad():
    outputs = model(**inputs)

pred = torch.argmax(outputs.logits, dim=1)[0].cpu().numpy()

# Create color-coded segmentation mask
np.random.seed(42)
color_map = np.random.randint(0, 255, (150, 3), dtype=np.uint8)
color_mask = color_map[pred]

# Create overlay
image_np = np.array(image.resize((pred.shape[1], pred.shape[0])))
overlay = (0.5 * image_np + 0.5 * color_mask).astype(np.uint8)

# Terrain-only extraction
terrain_classes = [13, 26, 29, 46, 52, 9, 16, 17, 21, 34, 68, 94]
terrain_mask = np.isin(pred, terrain_classes)
terrain_only = image_np.copy()
terrain_only[~terrain_mask] = 0

# Visualize results
plt.figure(figsize=(20, 5))

plt.subplot(1,4,1)
plt.imshow(image)
plt.title("Original")
plt.axis("off")

plt.subplot(1,4,2)
plt.imshow(color_mask)
plt.title("Segmentation (Semantic)")
plt.axis("off")

plt.subplot(1,4,3)
plt.imshow(overlay)
plt.title("Overlay")
plt.axis("off")

plt.subplot(1,4,4)
plt.imshow(terrain_only)
plt.title("Terrain Only (Hybrid Final)")
plt.axis("off")

plt.show()
```

The output will display a **4-panel comparison** showing:
- How the original barren landscape looks
- What terrain classes the model detected (color-coded)
- An overlay blending segmentation with the original
- The extracted terrain regions isolated from the rest of the image

## 🏜️ Best Use Cases

> **This model works best on desert terrains and barren landscapes.**

Since the dataset was generated from a **barren desert landscape** scenario in the Falcon AI simulator, the model has been specifically tuned for:

| Terrain Type | Performance |
|---|---|
| 🏜️ **Desert terrains** | ✅ Excellent |
| 🪨 **Barren / rocky landscapes** | ✅ Excellent |
| 🌾 **Arid grasslands** | ✅ Good |
| 🏔️ **Mountain terrain** | ⚠️ Moderate |
| 🌲 **Dense forest / urban** | ❌ Not recommended |

### Why?
The simulation environment in Falcon AI was configured as a **barren desert landscape**, meaning:
- The training images predominantly feature sand, dirt, rock, sparse vegetation, and open sky
- The model has learned terrain features specific to arid and barren environments
- For other terrain types (forests, cities, water bodies), the model may not generalize well without additional training data from those environments

### Improving Performance on Other Terrains
To extend the model to other terrain types:
1. Create new simulation scenarios in Falcon AI with different landscapes
2. Capture additional terrain images from those scenarios
3. Re-run the mask generation and training pipeline with the combined dataset
